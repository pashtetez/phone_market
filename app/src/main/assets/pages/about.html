MELT LCD version 1.5
    Основная задача приложения для смартфона (планшета) "МЭЛТ ЖКИ"- для ПОСТОЯННЫХ покупателей сделать УДОБНУЮ и БЫСТРУЮ выдачу информации по наличию и ценам на ЧАСТО ими покупаемые индикаторы и ИХ ЗАМЕНЫ при отсутствии на складе нужного товара. Из приложения нельзя заказать индикаторы, только посмотреть информацию.

    Нажатием на звёздочку в поле выбранного индикатора Вы вносите его в список "Избранное". Можно внести в список несколько моделей ЖКИ. В любой момент можно переключиться из полного списка товаров на "Избранное" нажав звёздочку в верхней части экрана.

    Нажатие на кнопку А предлагает выбор допустимых аналогов с различием только по цвету, чтобы можно было включать индикатор в ту же схему без перепаек. При этом предлагаются такие-же или лучшие по температурным характеристикам индикаторы.

    Строка поиска работает по нескольким буквам в названии ЖКИ, не обязательно писать его полное имя.

    Приложению требуется разрешение ТОЛЬКО на доступ в интернет, чтобы отслеживать актуальное состояние склада по сайту МЭЛТ www.melt.com.ru
